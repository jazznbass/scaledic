# Table with descriptive statistics

Table with descriptive statistics

## Usage

``` r
descriptives(data, round = 2, labels = FALSE)
```

## Arguments

- data:

  A data frame

- round:

  Digits for round function

- labels:

  Deprecated. Use [`rename_items()`](rename_items.md) instead.

## Value

A data frame with descriptive statistics
